# Quiz online by Naisya nurfalah azilla

projek ini adalah sebuah aplikasi  atau sebuah web kuis sederhana yang
di dalam nya menggunakan HTML, CSS dan juga JavaScript

## berikut cara menjalankan nya 
1. buka folder project 
2. kemudian jalankan file 'index.html' di 
browser atau di chrome

## fitur yang tersedia 
- menampilkan pertanyaan berupa pilihan ganda 
- dapat menghintung skor secara otomatis 
- dan dapat menampilkan status lulus atau tidak nya 

## keterangan 
dapat di katakan lulus jikalau skor >= 70.
